To get started with Source modding, you need to understand the basics. This VDC article will guide you through the fundamentals of modding.

  • When compiling, add Physgun.cpp to your server-side project and c_weapon_gravitygun.cpp to your client-side project (game/server/physgun.cpp and game/client/c_weapon_gravitygun.cpp respectively).
  • Avoid using VDC Lua; instead, use the Lua provided in our project's source code.
  • Do not use the source code of Combine Sandbox without explicit permission from the lead developer, @guestsneezeplayz.
  • The Python used is from This Guide
  • If you want to become a developer, join our Discord server and contact @guestsneezeplayz or @gmod13 / YourLocalMoon.
